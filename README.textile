Welcome to DividedJobs
DividedJobs is an enhanced version of play's Job.
Here you can set up as many job-pools as you can if necessary. 
By doing this, you can put your job into different job-pools.
It is useful in preventing job-pool from being hung-up for some job take long time to process.

And you can get more information about DividedJobs in "How to use DividedJob":https://github.com/mayan31370/DividedJob/wiki/How-to-use-DividedJob
